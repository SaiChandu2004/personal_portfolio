Insomnia Prediction

Developed a web application for identifying the risk of Insomnia using K-Means clustering to classify individuals based on sleep-related data. Used Python and Scikit-learn for data preprocessing, feature normalization, and model evaluation. Applied the same model to build a web application for easy user interaction, then used Matplotlib for data visualization.

Food Delivery App Interface

Built a Java-based console application simulating a food delivery system with features like restaurant selection, dynamic menu management, item customization, cart operations, and order processing. The project applied object-oriented programming concepts using interfaces, classes, and Java collections to ensure modularity. Users could add/remove items from the cart, apply customizations, calculate the total, and generate a detailed bill saved as a text file using Java File I/O.
